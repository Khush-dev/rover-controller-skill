Control the motion of any rover with voice commands

## About
This is for controlling the rover developed by iitb mars rover team. you can modify it for your own use case.

## Examples
* "Rover move forward"
* "Rover turn left"
* "Rover turn right"
* "Rover go ahead"
* "Kill (the|) rover"
* "Start (the|) rover"
